## 설명
해당 레포는 앞으로 진행할 기능 단위의 프로젝트의 boilerplate 입니다. 


## 기술스택
- client: Nextjs
- server: Spring boot
- db    : Postgre


## 실행방법
```

docker compose up -d --build

```
